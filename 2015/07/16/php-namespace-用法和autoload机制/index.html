php namespace 用法

若MESSAGE前不加入namespace，则默认会定义在global namespace下。

若要调用系统strlen函数，可用\strlen(“hello”);否则，会默认调用\test\strlen()，若未发现再在全局空间里寻找strlen。
常量和类名同理

相当于use a\b\c as c

配合__autoload魔术方法和spl_autoload_register方法可实现动态加载文件(lazy loading机制)

bool spl_autoload_register ([ callable $autoload_function [, bool $throw = true [, bool $prepend = false ]]] )

autoload_function欲注册的自动装载函数。如果没有提供任何参数，则自动注册 autoload 的默认实现函数spl_autoload()。(若绑定到类内的函数，第一个参数可以array('class_name','method_name')形式)。throw此参数设置了 autoload_function 无法成功注册时， spl_autoload_register()是否抛出异常。prepend如果是 true，spl_autoload_register() 会添加函数到队列之首，而不是队列尾部。`
